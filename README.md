## 控制台数独游戏

![](./images/game_start.png)

### 步骤

```bash
pip install sudokucli
```

## 选项

* `--coordinate/--no-coordinate` 开启坐标系，默认打开
* `--check/--no-check` 每一步进行合法检查，默认打开
* `--mode` 关卡难度，`easy`, `medium`, `hard`, `extreme`
